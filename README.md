# Skills

A collection of Claude skills for AI-enabled operators.

Each skill is a single SKILL.md file you drop into your Claude Project's skills folder to install. No setup beyond that.

## What's Here

| Skill | What It Does |
|---|---|
| [ai-work-ledger](./ai-work-ledger/) | Generates a structured career asset entry at the close of any AI work session. Includes data scrub, multi-platform export, and capability proof statement. |

## How to Install Any Skill

1. Navigate to the skill folder
2. Download `SKILL.md`
3. Drop it into your Claude Project's skills folder
4. Done — trigger phrases are in each skill's README
